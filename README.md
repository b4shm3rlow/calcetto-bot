## Setup
You can install or upgrade python-telegram-bot with:
```bash
 pip install python-telegram-bot --upgrade
```
Insert your API Token into the python file bot.py
```bash
 BOT_TOKEN_API = ''
```
For run locally:
```bash
 python bot.py
```
